• Фон сцены (космос) - обычное изображение
  • Земля - это сфера Mesh c текстурой в формате изображения
  • Земля вращается по всем 3 осям, независимо от скролла
  • Имитация звезд в пространстве - это массив Mesh из SphereGeometry
  • При скролле звезды движутся, а Земля постепенно приближается

  • Динозавр - это загруженная модель .glb с уже имеющейся там анимацией
  • Без текстур, чтобы меньше весил
  • Модель перемещается по направлению скролла

  • Астероид (или камень) - это модель с текстурой, но без анимации
  • Анимация вращения, как и у Земли, прописана руками, но только по 2 осям
  • Движение (и увеличение) происходит по формуле вычисления косинуса, в зависимости от скролла

  • Глобальное освещение состоит из 2 источников света - ambientLight и directionalLight
  • Написана поддержка ресайза окна браузера для адаптивности канваса
  • Используется самый популярный тип камеры - PerspectiveCamera
